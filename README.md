# TaskManager

A web application for managing personal tasks, built with Angular and Django REST Framework.

## Description

TaskManager allows users to create, edit, delete, and track their tasks. Each task has a title, description, status, priority, due date, and comments.

## Tech Stack

- **Frontend:** Angular 17+
- **Backend:** Django + Django REST Framework
- **Authentication:** JWT (SimpleJWT)
